# Лендинг для маникюра

Адаптивное одностраничное приложение для маникюрши. 

## Особенности
- Адаптивная верстка для мобильных устройств, планшетов и десктопов.
- Плавные анимации интерфейса благодаря библиотеке Framer Motion.
- Динамическая загрузка услуг и портфолио через JSON или API.
- Форма обратной связи с валидацией данных (Formik и Yup).
- Современный UI на базе Tailwind CSS.

## Используемые технологии
- React
- Tailwind CSS
- Framer Motion
- Formik и Yup
- Axios
- Lucide React

## Запуск проекта
1. Клонируйте репозиторий.
2. Установите зависимости: npm install
3. Запустите сервер разработки: npm start
